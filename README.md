# 🚫 AccessForbidden: AI Batch Upscaler

**AccessForbidden** is a robust, minimalist, and high-speed image upscaling tool designed for creators and developers who handle high-resolution digital assets. It specializes in processing large files (up to 45MB) with precision and speed.

## ✨ Key Features
* **Batch Processing:** Upscale multiple images simultaneously.
* **High-Res Support:** Specifically optimized for files between 10MB and 45MB.
* **Real-Time Tracking:** Live counter showing "X of Y images completed" during batch execution.
* **Multi-Format Export:** Download your upscaled files in PNG, JPEG, WEBP, TIFF and others Format
* **Performance:** Utilizes Lanczos interpolation for sharp, high-quality results.
* **Modular Design:** Built to easily integrate future tools like background removal and AI filters.

## 🛠️ Technical Stack
- **Language:** Python
- **UI Framework:** Streamlit
- **Processing Engine:** OpenCV & Pillow (PIL)
